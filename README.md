# Proyecto Portafolio - Micaela Agustina Martinez

Este es mi portafolio personal desarrollado para el curso de Frontend.

## Tecnologías Utilizadas
- **HTML5**: Estructura semántica.
- **CSS3**: Diseño responsivo con Flexbox y Grid, incluyendo un menú "Checkbox Hack" para móviles sin JavaScript.
- **Formspree**: Integración de formulario de contacto funcional.

## Secciones
- Header con navegación.
- Presentación (Hero) con foto de perfil.
- Grid de Proyectos.
- Habilidades (Python, JS, HTML, CSS).
- Frase (Quote) y Contacto.
